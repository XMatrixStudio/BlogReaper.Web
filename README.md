# BlogReaper.Web

## Install

```bash
$ git clone https://github.com/XMatrixStudio/BlogReaper.Web.git
$ npm --registry https://registry.npm.taobao.org install node-sass
$ npm i
```

PS: node-sass 可能需要淘宝源才能安装

### Run

```bash
$ npm run serve
```

### Vue ui

```bash
$ npm install -g @vue/cli
$ vue ui
```
